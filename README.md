# portfolio_2026 — 김천호 (Mack)

풀스택 소프트웨어 엔지니어 김천호(Mack)의 개인 포트폴리오 1페이지 사이트.

> AI를 가져다 쓰는 데서 멈추지 않고, 직접 환경을 설계·구축해 생산성과 운영 안정성을 함께 추구하는 백엔드 중심 엔지니어.

## Stack
- **Vite + React + TypeScript**
- **Tailwind CSS v4** (`@theme` 디자인 토큰)
- Source Serif 4 (display) · Inter (body)

## Develop
```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드 → dist/
npm run preview  # 빌드 미리보기
```
